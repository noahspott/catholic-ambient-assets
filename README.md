Reference images for the Catholic Ambient reel pipeline. See [catholic-ambient](https://github.com/noahspott/catholic-ambient) for the render pipeline.

## Structure

- `reference-images/vertical/` — 9:16 reel references (tall/centered subjects, space for text overlay)
- `reference-images/landscape/` — 16:9 long-form references (wide compositions, no overlay constraint)

Raw URL pattern:

```
https://raw.githubusercontent.com/noahspott/catholic-ambient-assets/main/reference-images/vertical/002-image-vertical-1.png
https://raw.githubusercontent.com/noahspott/catholic-ambient-assets/main/reference-images/landscape/005-image-main.png
```
